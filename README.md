# Printerest Generator

**Printerest Generator** is an application designed to create videos from images downloaded from Pinterest. It allows users to customize video length, image quantity, resolution, and add MP3 background music. The project is currently in development (INDEV) and may have some bugs or incomplete features.

## Features

- **Image Downloading**: Fetches images from Pinterest based on the input topic.
- **Image Cropping and Resizing**: Automatically crops and resizes images to fit the chosen resolution.
- **Video Creation**: Compiles images and adds MP3 background music to generate a video.
- **Resolution Selection**: Choose between different video resolutions.


## Requirements

To run this application, you'll need:

- Python 3.x
- Required Python libraries:
  - `tkinter`
  - `customtkinter`
  - `Pillow` (Python Imaging Library)
  - `moviepy` (for video editing)
  - `pinscrape` (for Pinterest scraping)
  - `webbrowser` (for opening links)
  - `shutil` (for file operations)
  - `subprocess` (for executing system commands)
